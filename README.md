Football Tactical AI

Uma prova de conceito utilizando Visão Computacional + Qwen
para gerar recomendações táticas a partir de um vídeo curto de futebol.

Objetivos

✔ Detectar jogadores
✔ Interpretar organização tática
✔ Gerar recomendações
✔ Produzir uma imagem tática automaticamente

Tecnologias

Python
OpenCV
YOLO
Qwen
Matplotlib
Google Colab
